## 大五人格推断

| 维度 | 推断值 | 置信度 | 观察依据 |
|------|--------|--------|---------|
| O 开放性 | 待观察 | - | 面对新方案/新工具时的态度 |
| C 尽责性 | 待观察 | - | 审查细节的严格度、对deadline的态度 |
| E 外向性 | 待观察 | - | 喜欢当面讨论还是文字指令 |
| A 宜人性 | 待观察 | - | 出错时的反馈方式 |
| N 神经质 | 待观察 | - | 压力下的情绪波动程度 |

*推断值范围: 低 / 中偏低 / 中 / 中偏高 / 高*
*置信度: 低（<3次观察）/ 中（3-10次）/ 高（>10次且一致）*
